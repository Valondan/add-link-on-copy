Этот плагин позволяет автоматически добавлять ссылку на сайт при копирование текста

== Описание ==

С помощью этого плагина вы можете автоматически добавлять ссылку на сайт когда-то кто копирует текст с вашего сайта.

прямо сейчас вы можете:

* Указать текст ссылки "Подробнее", по умолчанию стоит: "Источник:"
* Сколько тегов <br /> должно быть вставлено перед скопированным текстом и ссылкой или уведомлением об авторских правах
* Указать, хотите-ли включить текущий заголовок страницы в ссылку
* Сделать каждый скопированый тест только со ссылкой на главную страницу
* Решить, хотите-ли, чтобы название сайта было добавлено в ссылку
* Смотреть предварительные просмотры HTML и текстовых версий скопированного текста и ссылки

Я планирую расширить функциональность с помощью:
* Добавления подержки старых версий Internet Explorer
* Добавления других языков

== Установка ==

1. Загрузите каталог `added-link-on-copy` в `/wp-content/plugins/` вашей дириктории установки WordPress.
2. Активируйте плагин через меню "Плагины" в WordPress.

== Часто задаваемые вопросы ==

= Работает-ли плагин со всеми браузерами? =

Да, но может не работать с Internet Explorer версии ниже чем 10
